## Getting Started

1. **Clone the Repository**

    ```bash
    git clone https://github.com/DavittBarry/davitt-portfolio.git
    ```

2. **Navigate to the Directory**

    ```bash
    cd davitt-portfolio
    ```

3. **Install Dependencies**

    ```bash
    npm install
    ```

4. **Run Development Server**

    ```bash
    npm run dev
    ```
